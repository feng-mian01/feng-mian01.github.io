## 🚀 项目简介

这是一个系统的网络安全学习笔记仓库，涵盖从基础理论到实战技能的完整知识体系。采用**实验驱动**和**问题导向**的学习方法，每个知识点都配有实际操作和案例。

## 📚 学习体系

### 🎯 00-基础阶段
- **网络基础** - TCP/IP协议、HTTP/HTTPS、DNS解析等
- **操作系统** - Linux系统管理、Windows安全机制
- **编程基础** - Python自动化脚本、SQL查询语法

### 🔐 01-Web安全  
- **SQL注入** - 各种注入技术的原理与实战
- **XSS跨站脚本** - 反射型、存储型、DOM型XSS
- **CSRF攻击** - 跨站请求伪造的原理与防御
- **文件上传漏洞** - 文件类型检测绕过技术

### 🛠️ 02-渗透测试工具
- **Nmap** - 网络扫描与主机发现
- **Burp Suite** - Web应用渗透测试平台
- **Metasploit** - 漏洞利用框架

### 🏆 03-CTF实战
- 各类CTF比赛题解
- 日常练习记录
- 解题思路与技巧

1. **按顺序学习**：建议从00-基础阶段开始，逐步深入
2. **动手实践**：每个模块都包含实验步骤，请务必动手操作
