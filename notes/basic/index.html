网安基础入门 ​

万丈高楼平地起，网安学习从基础开始 🏗️

📌 核心学习内容 ​

1. 网络基础 ​

  • TCP/IP 协议族（IP/TCP/UDP/HTTP/HTTPS）
  • 端口与服务（常用端口：80/443/22/3389/3306等）
  • 网络分层模型（OSI七层/TCP/IP五层）
  • 后续更新：《TCP/IP 协议详解》笔记

2. Linux 基础 ​

  • 常用命令（ls/cd/grep/find/netstat等）
  • 权限管理（chmod/chown/sudo）
  • 文本编辑（vim/nano）
  • 后续更新：Linux 常用命令速查表

3. 编程基础 ​

  • Python 入门（语法/库/爬虫/脚本编写）
  • Shell 脚本（批量操作/自动化）
  • 后续更新：Python 网安常用库（requests/paramiko等）

📝 学习建议 ​

  1. 先理解网络基础，再学工具使用
  2. 多动手实操，不要只看理论
  3. 记笔记！记笔记！记笔记！

🎯 学习目标 ​

掌握网络基础概念，能熟练使用Linux常用命令，能用Python写简单的网安脚本。
