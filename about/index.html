About#

你好，我是 风 眠 。

这里是我的个人技术博客，也是我记录网络安全学习路径的地方。

从基础知识到实战练习， 从代码编写到系统研究， 我会把学习过程中的收获、踩坑、思考与复盘整理在这里。

What I Do#

  • Security Research
  • Linux Exploration
  • Programming
  • Technical Writing

Why This Blog#

技术需要积累，经验需要沉淀。

记录不是为了证明已经掌握， 而是为了留下不断前进的痕迹。

我希望这个博客既是自己的学习档案， 也是一个持续更新的安全实践笔记。

CTF & 靶场#

这些平台主要用来练习 Web、Pwn、Misc、逆向、提权与渗透测试思路，也是我平时学习和复盘的重要来源。

  • Hack The Box - 经典综合靶场，偏实战，适合提升渗透测试与综合分析能力。
  • TryHackMe - 路线清晰，对新手更友好，适合系统化学习。
  • VulnHub - 提供大量可下载的本地靶机，适合离线练习。
  • OverTheWire - 适合从 Linux 基础、提权与思维训练开始。
  • BUUCTF - 国内常用 CTF 平台，题型丰富，适合刷题与复盘。
  • CTFHub - 提供 Web、安全技能树与实战演练内容。

常用工具#

下面这些工具是我在学习、分析和日常排查中比较常用的一些在线工具。

  • CyberChef - 编解码、加解密、哈希、提取与数据处理的“瑞士军刀”。
  • IT-Tools - 聚合了许多常用开发与安全辅助工具，日常使用很方便。
  • Etherscan - 以太坊区块浏览器，可用于查看交易、地址、合约与链上数据。
  • 长亭百川云工具箱 - 提供编码转换、漏洞分析与安全辅助等常用在线工具。
  • Exploit Database - 查询漏洞利用、PoC 与安全研究资料。
  • JWT.io - 用于解析、调试和验证 JWT Token 的常用工具。
